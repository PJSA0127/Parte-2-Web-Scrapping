# Web Scraping - Quotes

## 📌 Descripción

Script en Python que realiza web scraping de frases desde una página pública.

## 🌐 Página utilizada

https://quotes.toscrape.com/

## 📊 Datos extraídos

* Frases
* Autores

## ⚙️ Instalación

```bash
python -m pip install -r requirements.txt
```

## ▶️ Ejecución

```bash
python scraping.py
```

## 📁 Resultado

Se genera un archivo:

quotes.csv

con los datos estructurados.
